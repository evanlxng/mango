About This Page!

My main ambition in building this website was to help people visualize and follow along with my journey in engineering.

I wanted to create a way for people to not just keep track of how far I've come as a student and an engineer, but also to inspire others to pursue and reflect on their own goals and acheivments!

In order to do that, I personally built this website from the ground up and entirely for free.
From the hosting and SSL certification, to the domain name and software used to draft and build these webpages,
I wanted to challenge myself to pursue the most cost effective method of building a website to show that it doesn't take much to build a platform on the internet and share your ideas with the world!

So how did I do it?

I used Figma and Google Drawings to draft the layout of the website and create all the figures and images you see on these pages, and then coded all the physical webpages using HTML, CSS, and JavaScript.

All the code for this website has been pushed from VSCode to a public Github repository that anyone can view as new content is added!*

As I progress through my academic and professional career, I hope to continue updating this website with any new information or accomplishments I attain along the way.
