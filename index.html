About This Page!

This personal website was built entirely from the ground up with the above programs!

I used Figma and Google Drawings to draft the layout of the website and create all the figures and images you see on the page, and then implemented the final design using HTML, CSS, and JavaScript.

All the code for this website has been pushed from VSCode to a public Github repository that anyone can view as new content is added.*

This website was also developed and run entirely for free. From the hosting and SSL certification, to the domain name and drafting software, I wanted to challenge myself to make the most cost effective format as a proof of concept that it doesn't take much to build a platform to share ideas with the world!

My main goal however in building this website was to help people visualize and follow along with my journey in engineering.

As I progress through my academic and professional careers, I hope to continue updating this website with any new information or accomplishments I attain along the way.
